# Random User Contacts App

I developed a contact android app using Random user api. This application provide to see all the users and the user details.

# Used Technologies
+ MVVM (Model - View - ViewModel)
+ Retrofit
+ Navigation Components
+ Kotlin Coroutines
+ Coil
